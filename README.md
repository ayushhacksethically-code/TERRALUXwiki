# TERRALUXwiki

Interactive Study Portal for **Cartographic Techniques & Thematic Mapping (GEOADS07T)** built using Hugo.

## Features
- **Modern UI**: Midnight dark mode, glassmorphic layout, dynamic neon glows, and custom typography.
- **Instant Search**: Client-side full-text search across all 25 syllabus topics.
- **Reading Mode**: Distraction-free content reading layout with automated reading-time estimates and sidebar navigation.

## How to run locally

To preview the website on your local machine:
1. Ensure you have [Hugo](https://gohugo.io/) installed.
2. Run the Hugo development server:
   ```bash
   hugo server
   ```
3. Open your browser and navigate to `http://localhost:1313`.
